#Taxi GS Website

professional website for my fathers taxi company

## Live
https://taxi-gs.de/

##features
- Booking form with automatic email notification
- real customer reviews powered by Firebase
- Mobile Hamburger Menu
- Fully responsive design for all devices

## technologies
-HTML, CSS, Javascript
-Firebase Firestore(database)
-Formspree (email handling)
-Netify(hosting)

##About
-Built for my father's taxi business to help him get more clients. 
-Built a separate password-protected invoice tool for the business owner:
- Generate professional PDF invoices with one click
- Monthly billing feature for corporate clients
- Auto-calculated VAT (7% / 19%)
- Auto-incrementing invoice numbers
- All company bank details pre-filled
